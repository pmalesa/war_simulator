# war_simulator
A small GUI app designed to annotating CT images.

## Virtual environment
### Setup
```bash
python -m venv war_simulator_env 
```

### Activation
```bash
source war_simulator_env/bin/activate
```

### Deactivation
```bash
source deactivate
```

## Install dependencies
```bash
pip install -r requirements.txt
```

## Application startup
```bash
python main.py
```
